# AI 分析页内容重构 设计文档

日期：2026-06-21
范围：个股「AI 分析」Tab 的内容与信息架构重构。基于「散户快速决策」的产品视角，删除重复内容、砍低价值模块、新增高价值内容，并完成排版统一。纯前端为主，距离%基准 bug 修复涉及前端计算。

用户基准：**散户快速决策** —— 打开页面只为快速判断「现在该买/持/卖」，内容服务此决策，不堆术语。

---

## 1. 背景

页面审计（基于真实模块与 AI 输出字段）发现：同一组交易点位讲了 3 遍、方向判断有 4 种表达、风险用 4 张评分卡堆术语，却唯独缺一句「现在该怎么做」和「AI 历史准不准」。页面冗长但决策力弱。

同时有 4 个排版问题待修：标识符位置不统一、AIVerdict 模块过长、风险/多时间框架风格不统一、详细诊断框套框。

## 2. 目标

- 删除重复内容（点位、R/R、方向、触发条件的多份表达）。
- 砍掉低价值模块（风险评分四卡、情绪 bar、装饰性置信度）。
- 新增高价值内容（「现在该怎么做」裁决行、距离%、命中率汇总）。
- 修复距离%基准 bug（应以当前价为基准）。
- 完成排版统一（标识符在卡外、拆分长模块、统一新模块、详细诊断去套娃）。

## 3. 设计（最终页面）

最终模块顺序（KEEP/MERGED/NEW/CUT）：

| 顺序 | 模块 | 处置 |
|------|------|------|
| 1 | **现在该怎么做（裁决行）** | NEW。一句祈使句，持仓/空仓两种视角。由 `immediate_action`+`trigger_condition`+`current_price` vs `entry/stop` 合成。 |
| 2 | **交易执行轴**（可视化轴 + 距入场/止损/目标%，基准=当前价） | MERGED。吸收顶部价格三卡与执行 6 卡，只留轴 + 距离%（T1/T2/仓位另列）。 |
| 3 | **核心研判摘要**（core_logic_summary） | KEEP。 |
| 4 | **计划生效/失效/加仓/复核**（Decision Brief 四卡） | MERGED。吸收 Header 触发副标题。 |
| 5 | **信心三维 + 总信心** | KEEP（砍情绪 bar）。无 breakdown 数据时降级为一行字。 |
| 6 | **一句话风险提示**（risk_level + max_drawdown） | NEW。替代整个 RiskAnalysis 四卡。 |
| 7 | **周期共振**（短/中/长 趋势 + 一句策略） | MERGED。砍 key_levels 三价位与置信度%。 |
| 8 | **关键假设断点** | KEEP（砍提示语空话）。 |
| 9 | **催化剂时间轴** | KEEP（默认折叠）。 |
| 10 | **我的持仓管理**（PositionOverlay） | KEEP。 |
| 11 | **Truth Tracker**（命中率汇总置顶 + 信心趋势，砍「当时价」列） | KEEP + NEW。 |
| — | 情绪偏差 bar / 执行策略横幅 / 风险回报卡 / RiskAnalysis 四卡+βRSI / Header 顶部价格三卡 | CUT。 |

排版统一（贯穿全部模块）：
- **标识符标准**：每模块 = 卡片外「竖色条 + 标题」(顶格) + 下方白底卡片内容。风险/多时间框架/拆出的新模块全部对齐。
- **AIVerdict 拆分**：拆为「AI 判研与交易计划」(裁决行+交易轴+核心摘要+Decision Brief+信心) 与原 Truth Tracker 独立模块；深度论据（情景/详细诊断）并入或保留为可折叠。
- **详细诊断去套娃**：外层一卡，内部三块（研判综述/结构化操作计划/多维逻辑表）去边框，改小标题+分隔线。

## 4. 实施批次（每批可独立交付验证）

**批次 A：删重复 + 砍低价值（减法，风险最低）**
- 砍：Header 顶部价格三卡、执行详情的风险回报卡与「执行策略」横幅、情绪偏差 bar、整个 RiskAnalysis 模块、MultiTimeframe 的 key_levels 三价位与置信度%、KeyAssumptions 提示语空话、Truth Tracker「当时价」列。
- 交付后页面更短、无重复，功能不减。

**批次 B：排版统一（4 点）**
- 标识符统一到卡外标题栏（风险已删，剩多时间框架 + 拆出的模块）。
- 拆分 AIVerdict 为两个带独立标题栏的模块。
- 详细诊断去套娃。

**批次 C：新增高价值内容（加法）**
- 「现在该怎么做」裁决行（持仓/空仓两视角）。
- 距入场/止损/目标 %（基准=当前价；修复 AIVerdict.tsx:953-957 的 entryMid 基准 bug）。
- 一句话风险提示（替代已砍的 RiskAnalysis）。
- Truth Tracker 命中率汇总置顶 + 信心趋势迷你折线。

## 5. 不做（YAGNI）

- 不改后端 AI 输出字段（全部用现有字段）。
- 不改 prompt（"AI 少给买卖动作"是另一议题）。
- 不动「标的信息」Tab。
- 删掉的模块组件文件保留在磁盘（仅移除渲染引用），便于回滚。

## 6. 验证

每批次：本地前端（3001）登录后看 GOOGL/MU AI 分析 Tab，`npx tsc --noEmit` + `npm run build` 通过。
- 批次 A：无重复点位/R/R/方向；风险四卡消失；页面明显变短。
- 批次 B：所有模块标识符在卡外；判研拆为两模块；详细诊断无嵌套框。
- 批次 C：裁决行出现且持仓/空仓文案正确；距离%以当前价为基准（手算核对一例）；Truth Tracker 顶部显示「N 次命中 M 次」。
