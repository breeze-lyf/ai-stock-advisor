## 一、后端 API（已完成）

### 1. 数据模型 - 7 张表

### 2. 服务层 - 5 个引擎
| 服务 | 功能 |
|------|------|

| `factor_research.py` | IC 分析、分层回测、换手率、衰减分析 |
| `portfolio_optimizer.py` | 6 种组合优化算法 |
| `quant_backtest.py` | 事件驱动回测 |
| `quant_signal.py` | 信号生成、风险管理 |

### 3. API 端点 - 15+ 个
| 路由 | 功能 |
|------|------|
| `GET/POST /factors` | 因子列表/创建 |
| `DELETE /factors/{id}` | 删除因子 |
| `GET /factors/{id}/ic-analysis` | IC 分析 |
| `GET /factors/{id}/turnover` | 换手率 |
| `GET /factors/{id}/decay` | 衰减分析 |
| `GET /factors/{id}/layered-backtest` | 分层回测 |
| `POST /optimize` | 组合优化 |
| `POST /backtest/run` | 执行回测 |
| `GET/POST /strategies` | 策略管理 |
| `POST /strategies/{id}/generate-signals` | 生成信号 |
| `POST /risk/check` | 风险检查 |

### 2. 功能模块

#### 模块 1：因子研究 (Factors)
**5 个子标签页：**

1. **因子列表**
   - 因子表格展示（名称、类别、IC 均值、ICIR、年化收益、夏普）
   - 创建自定义因子（名称、代码、类别、公式）
   - 删除自定义因子
   - 点击选择因子进行分析

2. **IC 分析**
   - 运行 IC 分析按钮
   - 4 个指标卡片：IC 均值、ICIR、T 统计量、样本数
   - IC 时序折线图

3. **换手率分析**
   - 平均换手率
   - 换手率时序数据

4. **衰减分析**
   - 初始 IC
   - 半衰期（天）
   - IC 衰减曲线

5. **分层回测**
   - 多空收益
   - 10 层分组收益对比
   - 权益曲线

#### 模块 2：组合优化 (Optimizer)
**6 种优化器：**
- 均值方差 (Mean-Variance)
- Black-Litterman
- 风险平价 (Risk Parity)
- 层次风险平价 (HRP)
- 最小波动率 (Min Volatility)
- 最大夏普 (Max Sharpe)

**优化结果展示：**
- 预期收益
- 波动率
- 夏普比率

#### 模块 3：策略管理 (Strategies)
- 策略列表展示
- 策略类型、调仓频率
- 一键生成交易信号

#### 模块 4：量化回测 (Backtest)
- 日期范围选择
- 运行回测按钮
- 回测结果展示：
  - 总收益、年化收益
  - 夏普比率、最大回撤
  - 波动率、胜率
  - 总交易数

#### 模块 5：风险管理 (Risk)
- 风险等级显示
- 持仓合规检查
- 风险指标监控

## 四、使用指南

### 访问路径
1. 启动后端：`cd backend && uvicorn app.main:app --reload`
2. 启动前端：`cd frontend && npm run dev`
3. 访问：`http://localhost:3000/quant`
4. 或点击导航栏"量化因子"标签

### 操作流程

#### 因子分析流程
1. 进入"因子研究"标签
2. 在因子列表中选择要分析的因子
3. 切换到"IC 分析"标签，点击"运行 IC 分析"
4. 查看 IC 指标和时序图
5. 切换到其他标签查看换手率、衰减、分层回测结果

#### 组合优化流程
1. 进入"组合优化"标签
2. 选择优化器类型（6 种）
3. 点击"运行优化"
4. 查看优化后的预期收益、波动率、夏普比率

#### 策略回测流程
1. 进入"量化回测"标签
2. 选择日期范围
3. 点击"运行回测"
4. 查看回测结果（收益、夏普、回撤等）
